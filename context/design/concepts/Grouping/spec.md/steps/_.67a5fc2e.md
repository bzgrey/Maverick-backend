# Grouping Concept

## Purpose

To manage the creation of groups and the membership of items within them.

## State

* `groups`: A set of unique `Group` identifiers.
* `members`: A mapping from a `Group` identifier to a set of `Item` identifiers.

## Actions

### createGroup

Creates a new, empty group.

* **Arguments:** none
* **Returns:** `g: Group` (the identifier for the newly created group)
* **Requirements:**
  * None.
* **Effects:**
  * A new, unique identifier `g` is generated.
  * `g` is added to the `groups` set.
  * An entry is added to `members` mapping `g` to an empty set.

### deleteGroup

Deletes an existing group and removes all its members.

* **Arguments:** `g: Group`
* **Requirements:**
  * The group `g` must exist in the `groups` set.
* **Effects:**
  * The group `g` is removed from the `groups` set.
  * The entry for `g` is removed from the `members` map.

### addMember

Adds an item to a group.

* **Arguments:** `g: Group`, `i: Item`
* **Requirements:**
  * The group `g` must exist in the `groups` set.
* **Effects:**
  * The item `i` is added to the set of members associated with group `g`.

### removeMember

Removes an item from a group.

* **Arguments:** `g: Group`, `i: Item`
* **Requirements:**
  * The group `g` must exist in the `groups` set.
  * The item `i` must be a member of group `g`.
* **Effects:**
  * The item `i` is removed from the set of members associated with group `g`.
